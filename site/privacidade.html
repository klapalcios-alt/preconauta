Política de Privacidade

A Liga Preconauta respeita sua privacidade. Este documento explica como coletamos, usamos e protegemos suas informações.

1. Dados coletados

Podemos coletar dados públicos de torneios (nome do jogador, resultados, decks e desempenho) e dados fornecidos pelos organizadores.

2. Uso dos dados

Os dados são usados para calcular rankings, estatísticas e publicar resultados da liga. Não vendemos ou compartilhamos dados para fins comerciais.

3. Cookies e armazenamento

O site pode usar armazenamento local apenas para preferências de visualização (ex.: tema claro/escuro).

4. Atualizações

Esta política pode ser atualizada quando necessário. A data de atualização fica no topo desta página.

5. Contato

Para dúvidas ou solicitações, use os links oficiais da Liga Preconauta.
